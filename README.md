# labtohub

Script to mirror GitLab repositories to GitHub

## Usage

1. Install [jq](https://stedolan.github.io/jq/) and curl;
2. Rename `config.sh.sample` to `config.sh` and change the settings;
3. Run `labhub.sh`.
